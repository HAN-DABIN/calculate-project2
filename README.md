# Calculator-project2
CH 2 계산기 과제

![img.png](img.png)

### ✅ 계산기 성능
1. **사칙 연산**(➕,➖,✖️,➗)을 할 수 있습니다.
2. 계산된 **결과값**을 볼 수 있습니다.
3. 계산된 값 중 첫 번째 결과를 <span style="color:red">삭제</span>할 수 있습니다.
4. 숫자를 입력하시면 **입력하신 숫자보다 큰 계산된 값**을 조회할 수 있습니다.

### 🚨 유의 사항
1. 계산 시 숫자로 입력해주십시오.
   <br>➡️문자로 입력 시 <span style="color:red">재입력</span>이 필요합니다. (예시: 하나, 둘)
2. 숫자를 0으로 나눌 수 없습니다.
   ➡️ 나눗셈 다음 숫자를 0으로 입력 시 <span style="color:red">재입력</span>이 필요합니다.
